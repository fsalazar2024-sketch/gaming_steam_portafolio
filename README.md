# 🎮 GAMING STEAM - Portafolio Gamer

![Steam Logo](assets/img/Steam_logo.png)

Portafolio personal inspirado en Steam que muestra juegos favoritos, estadísticas y enlaces a redes sociales. Desarrollado como parte del curso de programación web aplicando buenas prácticas de HTML/CSS y control de versiones con Git.

## ✨ Características

- **Diseño gamer moderno** con gradientes y efectos neon
- **Galería interactiva** de juegos con tarjetas animadas
- **Enlaces a redes sociales** (Steam, Twitch, Discord)
- **Diseño responsive** que se adapta a dispositivos móviles
- **Animaciones CSS** suaves con transiciones y efectos hover
- **Estructura semántica** con HTML5
